    React Status #282

    2022/4/2

    (opens new window)

    React 18.0 Released (opens new window) — Almost a year ago, the plan (opens new window) was for 18 to be simple to upgrade to with minimal changes needed – and despite the introduction of a new concurrent rendered, Suspense, new hooks, automatic batching, and more (all covered in this news post) they’ve pretty much nailed it. This React 18 upgrade guide (opens new window) covers the few things you need to consider, and if you’re a true completist, this changelog (opens new window) covers every tiny change.

    The React Team

    (opens new window)

    How to Build a Fullstack App from Scratch (opens new window) — Join Scott Moss for this detailed course looking at modern app building. Get to grips with a stack of tools, including the likes of Next.js, TypeScript, Postgres, and React, plus you'll learn about data modeling, authentication, state management, testing and more.

    Frontend Masters sponsor

    When Does React Render Your Component? (opens new window) — If you’re still a little flummoxed by React rendering behaviour, this takes another crack at explaining it by walking the fine line between superficial overview and technical deep dive.

    Zhenghao He

    ▶  The Story of Next.js (opens new window) — A good balance of detail and high level overview to tell the Next.js (opens new window) story. Despite being just 12 minutes long, it goes into a lot more background and history than you’d expect, too.

    Tyler McGinnis

    Remix: The Yang to React's Yin? (opens new window) — Kent C. Dodds continues to flesh out Remix (opens new window)’s raison d’etre with another insightful blog post: this time talking about React’s UI building yin as compared to Remix’s ‘Network Chasm’ yang, which he broadly defines as everything between the client and the server.

    Kent C. Dodds

    How to Use Props in React (opens new window) — We first linked to this three years ago but Robin has been updating his posts to modern standards, so it remains a fantastic resource now too and is packed with prop-oriented examples.

    Robin Wieruch

    # 😎 A neat RSS reader project

    Quite often, readers will write in to us (you can do this by hitting reply to any issue) to ask if we'll link to their latest React-powered project. We often grant this request but sometimes a project seems merely created with React, but isn't open source or particularly relevant in terms of a React newsletter. This week we have a nice exception!

    (opens new window)

    Reams (opens new window), by Adam Butler, is a React Native-powered RSS reader (remember those?) that uses AI-driven layouts to present stories in an immersive but individual style, and while it is available as an app, it's open source too (opens new window) so you can learn from the code if you want.
