    JavaScript Weekly #581

    2022/3/19

    (opens new window)

    Shader Park: Create Interactive 2D and 3D Shaders with JavaScript (opens new window) — An open source Web-based platform, community, and library (opens new window) for simplifying the mystifying world of shaders and GPUs by letting you create them procedurally with JavaScript. Lots to play with and explore (opens new window) here. Note: This site is heavy on the browser given its use of WebGL, so may not be suitable for every device.

    💡 If you'd prefer something solely in 2D on the good old HTML canvas, ▶️ this 55-minute introduction to creative, visual coding (opens new window) with JavaScript is a neat alternative.

    Prettier 2.6 Released (opens new window) — The popular code formatting tool now has a singleAttributePerLine option to only place one attribute per line (as is common in certain types of template), adds TypeScript 4.6 support, and better formats JS code in various contexts (e.g. decorators, inline await in JSX, end-of-line comments).

    How npm Packages Were Used in a Protest Against the War in Ukraine (opens new window) — A few days ago, users of projects reliant upon node-ipc (opens new window) (such as Vue CLI (opens new window)) began to notice odd behaviors, including files being overwritten on systems geolocated to Russia (or Belarus) and protest messages (opens new window). This is a complex story and another reminder of how fragile the package supply chain can be. The Register has a shorter and news-ier writeup (opens new window), if you prefer.

    Deno 1.20 Released (opens new window) — A pretty big release for the JavaScript / TypeScript runtime with increased performance, new commands for running benchmarks and test suites, timeouts for AbortSignal, plus an upgrade to V8 10.0 and TypeScript 4.6. If only all projects wrote release posts as thorough as this.

    # IN BRIEF:

    • Joel Smith brings us up to speed with what came out of GatsbyConf 2022 (opens new window).

    • 📅 Talking of events, VueConf US (opens new window) is headed to Fort Lauderdale, Florida between June 8-10, 2022.

    • Node.js has had a swathe of security releases (opens new window) (for each maintained branch) to fix an OpenSSL related issue of high severity (opens new window).

    • Boa (opens new window) is an in-progress JS lexer, parser and compiler being written in Rust. The newest version (opens new window) adds more ECMAScript features, supports almost 50% of the Test262 suite, and now uses a virtual machine to execute JS.

    • Writing Twitter-facing apps in JavaScript? Twitter has announced (opens new window) a new Twitter API v2 SDK (in beta) for TS/JS.
